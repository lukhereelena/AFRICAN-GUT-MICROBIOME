# MultiQC Report: South African Gut Metagenome QC

## Overview
This report summarizes quality control (QC) metrics for 10 South African gut metagenome samples (PRJNA678454).

## Samples
SRR17048881, SRR17048885, SRR17048888, SRR17048932, SRR17048935, SRR17048938, SRR17048941, SRR17048944, SRR17048947, SRR17048950

## Contents
- FastQC: Read quality, GC content, adapter content
- Fastp: Filtering stats, Q20/Q30 rates, duplication rates
